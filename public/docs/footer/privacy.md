# Privacy and data usage

This website is part of a non-commercial research project developed in the context of doctoral study.

This site does not collect personal data, use cookies, or track user behaviour for analytics or advertising purposes.

No identifiable personal information is stored, shared, or sold by this site. If you choose to get in touch, please use the [Contact form](/contact). Your details will only be used to respond to your message and will not be stored or shared for any other purpose.
